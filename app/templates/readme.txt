== Description ==
This plugin imports Webflow CMS content into WordPress with progress tracking and automatic field mapping.

== Features ==
- Automatic import on activation
- Real-time progress tracking
- Detailed import results
- Support for all Webflow field types
- ACF Pro integration

== Installation ==
1. Install and activate Advanced Custom Fields Pro
2. Upload this plugin to your WordPress site
3. Activate the plugin - import will run automatically
4. Check admin notices for import results

== Frequently Asked Questions ==
= Why isn't my content appearing? =
Make sure you have ACF Pro installed and the field names match your Webflow collection fields.

= How can I re-run the import? =
Go to Tools > Webflow Import and click "Run Import Now"
